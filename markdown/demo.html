1.1 标题

# 一级标题 ## 二级标题 ### 三级标题 #### 四级标题 ##### 五级标题 ###### 六级标题

1.2 列表

1.2.1 无序列表
在文字前加上 - 或 * 即可；- * 等符号要和文字之间加上一个字符的空格！
1.2.2 有序列表
直接在文字前加1. 2. 3. 符号；

1.3 引用

需要在文本前加入 > 这种尖括号（大于号）即可；
Tips： >符号要和文字之间加上一个字符的空格！而且是在英文输入法的条件下输入！

我一直在使用引用。

1.4 链接

如果是图片的话就要将图片的本地文件放到与该md文件相同的路径下，且填写文件的相对路径
如果是网页链接就直接贴上网页链接，http一定要有
注意文件名最好不要带中文，好像不行。还有，如果创作的文章需要发布到网上，需要将图片上传才可以！

使用相对路径插入图片。比如你把一个叫做1.png的图片和*.md文件放在一起，那么你就可以用这种方式插入图片：![](1.png) 
如果不想放在同一层级,那么就可以这样插入:![](foldername/1.png) 表示引用同层级一个叫做”foldername”的文件夹中的1.png图片,以此类推

1.5 字体

用两个 * 包含一段文本就是粗体的语法，用一个 * 包含一段文本就是斜体的语法

加粗 斜体 斜体加粗

1.6 代码框

连续三个反引号`，就是按键1前面的那个符号再加代码类型
如果不行则需要设置其他模式才能实现代码样式，操作如下：tools->options->markdown->markdown processor->commonmark。至此OK！或者设置为GitHub离线格式也行。

for i in range(5):
    print(i)

for i in range(5)

1.7 分割线

分割线的语法只需要三个 * 号。

2 高级玩法

2.1 按F6浏览器打开，代码高亮显示

MarkDownPad 2 默认不会对代码块进行着色，要达到酷炫的代码高亮的效果，需要进行一些配置。
进入markdownpad2》高级》工具》选项》高级》HTMLHead编辑器 
将如下代码粘贴进去

<link rel="stylesheet" href="https://cdn.bootcss.com/highlight.js/8.0/styles/solarized_dark.min.css">
<script src="https://cdn.bootcss.com/highlight.js/8.0/highlight.min.js"></script>
<script>hljs.initHighlightingOnLoad();</script>

2.2 自定义样式表

MarkDownPad 2内置的样式表有：markdown.css 、none.css 、markdownpad-github.css、solarized-dark.css 、solarized-light.css

选择“工具——选项——样式表”可以看到样式表的设置界面 或“按F7——样式表”

3 快捷键

Ctrl + B 粗体
Ctrl + I 斜体
Ctrl + Q 引用
Ctrl + K 代码
Ctrl + 1 标题1
Ctrl + 2 标题2
Ctrl + 3 标题3
Ctrl + 4 标题4

Ctrl + R 水平标尺
Ctrl + T 时间戳
F4 ： 启用水平布局/竖直预览
F5 ： 启用/关闭实时预览
F6 ： 在浏览器中预览
